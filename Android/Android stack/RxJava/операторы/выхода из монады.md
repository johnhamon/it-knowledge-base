Монады - это своего рода абстрактные конструкторы типов данных, которые инкапсулируют логику программы вместо данных в доменной модели.

`Observable` - это монада. Rx-код декларирует, что нужно сделать, но фактическая обработка происходит не при выполнении Rx-операторов, а при испускании значений.
##### BlockingObservable
Первый шаг к блокирующему получению данных из `Observable` переход к `BlockingObservable`. Любой `Observable` может быть преобразован в `BlockingObservable` одним из двух способов: 
```java
public final BlockingObservable<T> toBlocking()
```
```java
public static <T> BlockingObservable<T> from(Observable<? extends T> o)
```
BlockingObservable не расширяет Observable и не может быть использован с нашими обычными операторами Rx. У него есть собственные реализации небольшого набора функций, которые позволяют извлекать данные из Observable блокирующим образом.
